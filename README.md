# react-chat

웹소켓 기반 채팅 시뮬레이터 개발

- WebSocket을 활용한 실시간 채팅 시스템 구현
- 다수의 가상 사용자 생성 및 채팅 시스템
- React + Spring Boot 기반
- AWS EC2 서버 배포 및 Jenkins를 통한 CI/CD 자동 빌드 구성
- Codex를 활용한 개발 생산성 향상
