# react-chat

웹소켓 기반 채팅 시뮬레이터 개발

- WebSocket을 활용한 실시간 채팅 시스템 구현
- React + Spring Boot 기반 구조 설계
- AWS EC2 서버 배포 및 Jenkins를 통한 CI/CD 자동화 구축
- 도메인 구매 및 Cloudflare DNS/SSL 설정을 통한 서비스 도메인 연결
- Codex 활용
